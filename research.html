Research Highlights

Block Compressed Sensing (BCS-SPL)
Research on image and video compression and restoration using compressed sensing techniques. Developed Smoother Projected Landweber (SPL) reconstruction methods for efficient multiscale block compressed sensing.
[ Project Website ]
MEAM-BO (Modified Embedded-Atom Method with Bond Order)
Development of parallel molecular dynamics simulation codes and interatomic potentials for organo-metallic composite systems, polymers, and hydrocarbons. Implemented within the LAMMPS framework to enable large-scale predictive modeling.
[ GitHub Repository ]
Design Optimization Tool (DesOptTool)
Machine learning-based surrogate modeling and design optimization frameworks for complex engineering applications, including directed energy deposition and structural protective designs.
[ GitHub Repository ]
